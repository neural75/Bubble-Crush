Welcome to Bubble Crush.

This is an open source project involving funny balls that collide! yeee. It is derived from Bubble Mark test benchmark and aims to test javascript+HTML5 application on smartphones in a cross platform environment such as Phonegap. Then some sort of user interaction will be added, crushing the bubbles of course and also some game play would be a nice to have feature.

For now it will only display the simulation (it uses the DHTML method to move the balls) but I wanna change that to use canvas and other html5 goodies with the help of Createjs framework, also to add sound effects and other things. Stay tuned.

Download

You can find the last build for smartphones Here.

Currently it's working only for Android devices (enabling apk download in your phones) but you can tell me differently if you have blackberries or winphones.

Note: IOS is not supported right now because I don't want to publish it in the App Store, just test the application with regular download. Also a note for Windows Phones: they tell me you have to push the package in developer mode.
